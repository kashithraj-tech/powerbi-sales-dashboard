# Power BI Sales Dashboard

## Project Overview
This project is an interactive Sales Dashboard created in Power BI to analyze sales performance, segment trends, and sub-category breakdowns. It provides actionable insights to help identify growth opportunities.

## Key Features
* **Sales Analysis:** Visualization of total sales across different segments.
* **Sub-Category Insights:** Detailed breakdown of sales performance by sub-category.
* **Interactive Filtering:** Ability to filter data for deeper analysis.

## Tools & Technologies
* **Power BI Desktop**
* **GitHub** (for version control)

## How to View
To view this dashboard:
1. Download the `dashboard.pbix` file from this repository.
2. Open the file using Microsoft Power BI Desktop.
